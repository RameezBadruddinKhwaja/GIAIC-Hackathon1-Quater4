# Welcome to Physical AI & Humanoid Robotics

This interactive textbook teaches you how to build AI-powered humanoid robots using modern frameworks like ROS 2, NVIDIA Isaac Sim, and Vision-Language-Action (VLA) models.

## What You'll Learn

- **Module 1: ROS 2 Fundamentals** - Nodes, Topics, and the ROS 2 ecosystem
- **Module 2: Robot Simulation** - Gazebo and URDF for robot modeling
- **Module 3: Advanced Simulation** - NVIDIA Isaac Sim and Nav2 navigation
- **Module 4: AI for Robotics** - Vision-Language-Action models and voice control
