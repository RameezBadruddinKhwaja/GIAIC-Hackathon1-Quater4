# notebooklm-slides Skill

## Purpose
Generate presentation slides and study materials (optional skill).

## Key Patterns

### Slide Structure
```markdown
## Presentation: Week {N} - {Topic}

### Slide 1: Title
**{Topic Name}**
Physical AI & Humanoid Robotics Textbook
Week {N}

### Slide 2: Learning Objectives
By the end of this lesson, you will:
- {Objective 1}
- {Objective 2}
- {Objective 3}

### Slide 3: Key Concept
**{Concept Name}**

{Definition}

{Analogy or visualization}

### Slide 4: Code Example
```{language}
{Code snippet with comments}
```

### Slide 5: Architecture Diagram
{Mermaid diagram or image reference}

### Slide 6: Key Takeaways
- ✅ {Takeaway 1}
- ✅ {Takeaway 2}
- ✅ {Takeaway 3}

### Slide 7: Next Steps
{Preview of next lesson}
{Resources for further learning}
```

### NotebookLM Integration
- Export chapter content to NotebookLM
- Generate study guide
- Create flashcards
- Produce audio overview

## Usage Context
- Lecture preparation
- Study materials
- Quick reference
- Audio learning
