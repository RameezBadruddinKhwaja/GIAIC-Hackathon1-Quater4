# Skill: Docusaurus Formatting

## Admonitions
Use Docusaurus specific admonitions for notes and warnings:

:::note
This is a standard note.
:::

:::tip
Pro tip for the reader.
:::

:::danger
Critical warning.
:::

## Code Blocks
Always specify the language and file title if relevant:

```python title="main.py"
print("Hello World")
````

## Tabs

Use tabs for showing multiple options (e.g., Python vs JS):

\<Tabs\>
\<TabItem value="py" label="Python"\>...\</TabItem\>
\<TabItem value="js" label="JavaScript"\>...\</TabItem\>
\</Tabs\>
