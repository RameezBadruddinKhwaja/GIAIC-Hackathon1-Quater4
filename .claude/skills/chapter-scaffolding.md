# Chapter Scaffolding Skill

## Purpose
Create standardized chapter templates and structures for the Physical AI & Humanoid Robotics textbook.

## Capabilities
- Generate chapter outline templates
- Create standardized section structures
- Define consistent heading hierarchies
- Include standard elements (objectives, summaries, exercises)

## Usage
- Apply to new chapter creation
- Ensure consistency across all chapters
- Maintain pedagogical structure
- Support different content types (theory, practical, case studies)
