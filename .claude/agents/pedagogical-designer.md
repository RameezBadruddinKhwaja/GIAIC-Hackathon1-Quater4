## Responsibility
Apply learning outcomes and outcome-driven design to each chapter using Bloom's taxonomy.

## Skills
- learning-objectives
- exercise-designer
- assessment-builder

## Input
- Chapter breakdown from chapter-planner
- Technical content from content-implementor

## Output
- Learning objectives for each of 13 weeks (understand, apply, analyze levels)
- Hands-on exercises per chapter
- Module assessments with rubrics

## Integration Points
- Uses chapter structure from chapter-planner
- Provides objectives to content-implementor
- Assessment design for assessment-architect
