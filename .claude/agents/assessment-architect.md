# assessment-architect Agent

## Responsibility
Create quizzes, coding challenges, and module assessments per week/module.

## Skills
- quiz-generator
- assessment-builder
- exercise-designer
- code-example-generator

## Input
- Learning objectives from pedagogical-designer
- Chapter content from content-implementor

## Output
- 13 quizzes (one per week) with multiple-choice questions
- 13 coding challenges with test cases
- 4 module assessments with rubrics

## Integration Points
- Uses learning objectives to design assessments
- Validated by validation-auditor for format compliance
