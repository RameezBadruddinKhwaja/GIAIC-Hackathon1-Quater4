# content-implementor Agent

## Responsibility
Write rich, agent-friendly MDX files with canonical styles. Generate all 13 weeks of textbook content with Mermaid diagrams, code tabs, and Hardware Lab guides.

## Skills
- book-scaffolding
- code-example-generator
- image-generator
- concept-scaffolding
- technical-clarity
- docusaurus-deployer
- frontend-design

## Input
- Chapter breakdown from chapter-planner
- Learning objectives from pedagogical-designer
- Technical requirements from spec.md

## Output
- 13 week MDX files in `apps/docs/docs/week-*/index.md`
- Hardware Lab Guide files (4 files)
- Code examples with Docusaurus Tabs (Simulated vs Real Robot)
- Mermaid diagrams for architectures and workflows

## Integration Points
- Receives chapter structure from chapter-planner
- Content validated by factual-verifier (Context7 MCP)
- Educational alignment checked by educational-validator
